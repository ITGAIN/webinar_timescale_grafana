# webinar_timescale_grafana
Needed files for the itgain webinar for timescale and grafana database observability

1) Execute all SQL Statements in webinar-timescale-grafana.sql to get data into sample tables (Note: one command is for the shell of the timescale container)
2) Create a new dashboard and use the sql in seminar-dashboard-sql.sql to show how dashboards can be created
3) Use Workshop Result.json for importing a new dashboard. this boards has all panels for the sql in the sql example


Note: Storage is persistent in current directory in folder ".data"

Note: Grafana 7 can be used to show a query builder works, grafana 12 for fresh version of grafana (with a query builder that is worse than in grafana 7)
